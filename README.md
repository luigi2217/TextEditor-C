# Text Editor

A text editor built using generalized lists, written in C.

## Description

This project implements a text editor based on generalized lists, a data structure commonly studied in Data Structures courses.

It was developed as part of the Data Structures coursework.

## Features

- Text editing using generalized lists
- Insert and remove text
- Load text from a file
- Save edited text

## Project Structure

- `editor.c` — Main program
- `listaGeneralizada.h` — Generalized list implementation
- `listaGeneralizada.c` — Generalized list functions
- `exemplo.txt` — Sample text file for testing (written in Portuguese)

## Development Environment

Developed and tested using Dev-C++.

## Language

The sample file `exemplo.txt` is written in Portuguese.
